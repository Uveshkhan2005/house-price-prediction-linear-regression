# 🏠 House Price Prediction (Linear Regression)

This project is a simple machine learning model that predicts house prices using **Linear Regression** based on features like area and number of rooms.

---

## 📌 Project Overview

In this project, we:
- Loaded the dataset
- Performed basic data preprocessing
- Selected important features
- Trained a Linear Regression model
- Predicted house prices
- Evaluated model performance

---

## 📊 Dataset

The dataset contains house-related information such as:
- Area (size of the house)
- Number of rooms
- Price (target variable)

---

## ⚙️ Steps Performed

### 1. Data Loading
- Loaded dataset using Pandas

### 2. Data Preprocessing
- Checked dataset structure
- Handled data 

### 3. Feature Selection
- Selected relevant input features (e.g., area, rooms)

### 4. Model Training
- Used **Linear Regression** from Scikit-learn

### 5. Prediction
- Predicted house prices using the trained model

### 6. Model Evaluation
- Evaluated model using metrics like:
  - Mean Absolute Error (MAE)
  - Mean Squared Error (MSE)
  - R² Score

---

## 🤖 Model Used

- Linear Regression
